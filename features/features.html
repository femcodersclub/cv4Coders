Plantilla Minimalista

La plantilla minimalista está diseñada para destacar tu perfil profesional con claridad y elegancia. Hemos agregado comentarios directamente en el código para que sea más sencillo de personalizar. Sigue estos pasos para utilizarla:

  • Clona el repositorio desde GitHub.
  • Abre el archivo en tu editor de texto favorito.
  • Edita la plantilla según tus necesidades y agrega tu información personal.
  • Una vez que estés satisfecho con el resultado, utiliza la funcionalidad integrada para descargar el archivo en tu dispositivo.
