Documentación del Proyecto CV4Coders

Este proyecto ha sido creado con la finalidad de proporcionar plantillas de currículum vitae (CV) diseñadas especialmente para la comunidad femCoders Club. Todas las plantillas están desarrolladas utilizando HTML, CSS y JavaScript.

Invitamos a toda la comunidad a contribuir creando nuevas plantillas y enviando un pull request.

Todas las plantillas son personalizables: hemos incluido comentarios en el código para facilitar su edición. También puedes clonar el proyecto, editarlo según tus necesidades y probar la opción básica de personalización directamente desde el navegador.

Documentación de Plantillas

Plantilla Minimalista

Plantilla de CV minimalista y elegante, diseñada para una presentación profesional de tu perfil. Incluye sistema de edición en el navegador y exportación a PDF.

Características

  • Editor en vivo integrado
  • Exportación directa a PDF
  • Diseño responsive
  • Personalización de colores, fuentes y contenido.

Estructura del Proyecto

  • HTML: Contiene secciones clave y cada sección incluye elementos con el atributo contenteditable="true" para permitir la edición en tiempo real.
  • CSS: Uso de grid
  • JavaScript: Funciones para exportar a PDF con la biblioteca html2pdf.js..

Plantilla Creativa

La Plantilla de CV Creativo está diseñada para profesionales tecnológicos que deseen destacar su perfil mediante un diseño único y moderno. Este CV es completamente personalizable y puede descargarse en formato PDF, además de incluir características más avanzadas como la subida de una foto de perfil.

Características

  • Editor en vivo integrado
  • Exportación directa a PDF
  • Diseño responsive
  • Personalización de colores, fuentes y contenido.
  • Subida de foto de perfil
  • Interfaz Creativa y Moderna

Estructura del Proyecto

  • HTML: Contiene secciones clave y cada sección incluye elementos con el atributo contenteditable="true" para permitir la edición en tiempo real. Uso de etiquetas semánticas y comentarios para facilitar la navegación en el código.
  • CSS: Diseño basado en grid y flexbox para una distribución dinámica.
  • JavaScript: Función para subir y previsualizar fotos de perfil y descarga de PDF configurada con opciones personalizables.

Plantilla Técnica

La plantilla técnica emula un entorno de desarrollo en Visual Studio Code, diseñada específicamente para desarrolladores que deseen presentar su experiencia de una manera única y técnica. Incluye secciones editables, un terminal interactivo y la capacidad de descargar el CV como PDF.

Características

  • Diseño Inspirado en un IDE
  • Editor en vivo integrado
  • Exportación directa a PDF
  • Diseño responsive
  • Personalización de colores, fuentes y contenido.
  • Terminal Interactiva

Estructura del Proyecto

  • HTML: Contiene secciones clave y cada sección incluye elementos con el atributo contenteditable="true" para permitir la edición en tiempo real.
  • CSS: Estilo de Terminal y Editor, transiciones suaves en barras de progreso y tarjetas.
  • JavaScript: Terminal simulada, descarga de PDF, navegación entre Secciones, pestañas activas

Instrucciones de Uso

  • Edición del Contenido:Haz clic en cualquier campo editable marcado con contenteditable="true" y modifica los textos directamente en el navegador.
  • Uso del Terminal: Introduce comandos básicos como help, profile, clear.
  • Descarga del CV en PDF
  • Navegación por el CV: Haz clic en las pestañas de navegación para desplazarte por las diferentes secciones del CV.

Personalización

  • Colores: Modifica los colores de la plantilla en el archivo CSS.
  • Agregar Nuevos Comandos: Personaliza la terminal añadiendo nuevos comandos en JavaScript 
    if (command === 'newcommand') {
                                    output.textContent += '\nRespuesta personalizada para newcommand\n';
                                }
  • Contenido: Añade o elimina secciones según tus necesidades.
